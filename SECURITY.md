# Security Policy

## Supported versions

Security fixes are provided for the latest `1.x` release line.

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting for this repository. Do
not open a public issue for a suspected vulnerability. Include the affected
version, reproduction steps, impact, and any suggested mitigation.

CorvidLabs will acknowledge a report within three business days and coordinate
disclosure after a fix or mitigation is available.
