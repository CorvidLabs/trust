# Research

SemVer 2.0 forbids leading zeroes in numeric major, minor, patch, and numeric
prerelease identifiers; build identifiers may contain numeric leading zeroes.
Python `Path.as_uri()` percent-encodes spaces, so rejecting every percent sign
also rejects canonical safe local paths. URL parsing must decode only after
rejecting encoded `/` and `\\`, then apply the existing resolved-path boundary
and symlink checks.

The generated workflow is public guidance and intentionally uses the maintained
`v1` channel. Trust's nested SpecSync, Augur, Attest, Atlas, checkout, upload, and
deploy dependencies remain immutable. Organization rollout policy is stricter
and pins generated consumer workflows to reviewed release commits.
